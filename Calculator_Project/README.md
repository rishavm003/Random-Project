# Calculator Project

A versatile calculator application featuring both a graphical user interface (GUI) and a command-line interface (CLI) for performing mathematical operations.

## 🛠 Technology Stack
- **Language**: Python 3.x
- **GUI Framework**: Tkinter (Standard Python library)
- **CLI**: Standard input/output

## 📐 System Design
The project follows a modular design pattern:
- **Logic Layer**: Mathematical computations and input parsing.
- **Presentation Layer**: 
  - `gui.py`: Provides a modern button-based interface.
  - `cli_version.py`: Provides a lightweight terminal-based interaction.

## 🚀 How to Start

### GUI Version
```bash
python gui.py
```

### CLI Version
```bash
python cli_version.py
```
