# To-Do List Manager

A task management application to help you organize your daily activities. It supports adding, deleting, and marking tasks as complete.

## 🛠 Technology Stack
- **Language**: Python 3.x
- **GUI Framework**: Tkinter
- **Persistence**: Text file storage (`tasks.txt`)

## 📐 System Design
- **CRUD Operations**: Create, Read, Update, and Delete functionality for tasks.
- **File I/O**: Syncs task state to a local file to ensure data is saved between sessions.
- **UI State**: Dynamically updates the list view when tasks are modified.

## 🚀 How to Start

### GUI Version
```bash
python gui.py
```

### CLI Version
```bash
python main.py
```
