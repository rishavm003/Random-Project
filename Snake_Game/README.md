1: # Snake Game (Modern Edition)
2: 
3: A classic Snake game with two versions: a premium web-based GUI and a lightweight console application.
4: 
5: ## Versions
6: 
7: ### 1. Modern Web GUI (Recommended)
8: A premium, neon-inspired web application with smooth animations and interactive effects.
9: - **Location**: `gui/index.html`
10: - **Features**: Neon graphics, 12/24h toggle, high-score persistence, and arrow key support.
11: 
12: ### 2. Classic Console Version
13: A lightweight C-based application for Windows.
14: - **Location**: `SnakeGame.exe`
15: - **Features**: Minimal footprint, flicker-free rendering.
16: 
17: ## GUI Features
18: 
19: - **Neon Visuals**: Glowing snake and food with a modern dark theme.
20: - **High Score**: Saves your best score locally in the browser.
21: - **Responsive Controls**: Supports both **Arrow Keys** and **WASD**.
22: - **Glassmorphism**: Sleek UI panels for stats.
23: 
24: ## How to Run
25: 
26: ### Web GUI
27: 1. Navigate to the `gui/` folder.
28: 2. Open `index.html` in any modern browser.
29: 
30: ### Console Version
31: 1. Compile `main.c` using `gcc main.c -o SnakeGame.exe`.
32: 2. Run `.\SnakeGame.exe`.
33: 
34: ## Controls
35: 
36: - **Arrow Keys / WASD**: Change direction.
37: - **Space / Enter**: Start/Restart game (Web).
38: - **X**: Exit (Console).
