# Number Guessing Game

An interactive game where the computer selects a random number and the player tries to guess it with helpful "Higher" or "Lower" hints.

## 🛠 Technology Stack
- **Language**: Python 3.x
- **GUI Framework**: Tkinter
- **Randomization**: Python `random` module

## 📐 System Design
- **Random Engine**: Generates a target number within a specified range (e.g., 1-100).
- **Feedback Loop**: Compares user input to the target and provides directional feedback.
- **Score Tracking**: Keeps track of the number of attempts taken to win.

## 🚀 How to Start

### GUI Version
```bash
python gui.py
```

### CLI Version
```bash
python main.py
```
