# Countdown Timer

A sleek countdown timer application that allows users to set a time and watch it count down with visual updates.

## 🛠 Technology Stack
- **Language**: Python 3.x
- **GUI Framework**: Tkinter
- **Time Management**: Python `time` and `threading` modules

## 📐 System Design
- **State Management**: Tracks remaining seconds and active status.
- **Background Threading**: Uses threading to ensure the GUI remains responsive while the timer is running.
- **Validation**: Ensures users enter valid numerical time values.

## 🚀 How to Start

### GUI Version (Recommended)
```bash
python gui.py
```

### CLI Version
```bash
python main.py
```
